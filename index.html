同期設定

Supabase でデバイス間同期（無料）

セットアップ手順
  1. supabase.com で無料アカウント作成
  2. 新しいプロジェクトを作成
  3. SQL Editor で実行:
    CREATE TABLE tasks (id TEXT PRIMARY KEY, title TEXT, category TEXT, priority TEXT, due TEXT, done BOOLEAN DEFAULT FALSE, subtasks JSONB DEFAULT '[]', created_at BIGINT, updated_at BIGINT);
    ALTER TABLE tasks ENABLE ROW LEVEL SECURITY;
    CREATE POLICY "allow_all" ON tasks FOR ALL USING (true);
    ALTER PUBLICATION supabase_realtime ADD TABLE tasks;
  4. Settings > API から URL と anon key をコピー
